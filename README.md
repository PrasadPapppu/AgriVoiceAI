# 🎙️ VoiceBot Project

An AI-powered voice assistant that can record audio, process speech, and generate intelligent responses in real-time.

---

## 🚀 Features

* 🎤 Real-time voice recording
* 🧠 Speech-to-text processing
* 💬 AI-generated responses
* 🔊 Text-to-speech playback
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js / Express *(or Python if applicable)*
* APIs: Speech Recognition + AI APIs
* Other: Web Audio API

---

voicebot-project/

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/voicebot-project.git
cd voicebot-project
```

### 2. Install dependencies

```
npm install
```

### 3. Add environment variables

Create a `.env` file and add:

```
API_KEY=your_api_key_here
```

### 4. Run the project

```
npm run dev
```

## ▶️ Usage

1. Click on the microphone button
2. Speak into your device
3. Wait for processing
4. Get AI-generated response
